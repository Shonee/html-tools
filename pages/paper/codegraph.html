CodeGraph

预索引代码知识图谱（Pre-indexed Code Knowledge Graph）— 让 AI Agent 减少 58% 工具调用，提速 22%

MIT TypeScript 26 种语言 8 种 Agent 100% 本地

解决什么问题

AI Agent 每次理解代码都重复 grep/read 逐文件爬取，浪费 60-70% Token 预算。CodeGraph 预索引图谱让 Agent 一次调用拿到精准上下文。

核心特性

●
精准上下文— 一次调用返回符号源码 + 调用链 + 影响半径
●
自动同步— OS 文件事件 + 2s 防抖，改代码图谱即更新
●
影响分析— 修改前追踪 callers/callees 及完整影响半径

框架路由— 17 种 Web 框架 URL→Handler 映射

跨语言桥接— Swift↔ObjC · RN Bridge · Expo Modules
●
100% 本地— 无 API Key，SQLite，可选 codegraph.json 配置

完整使用指南

1 安装 CLI

# macOS / Linux（无需预装 Node.js）curl -fsSL https://raw.githubusercontent.com/colbymchenry/codegraph/main/install.sh | sh# Windows (PowerShell)irm https://raw.githubusercontent.com/colbymchenry/codegraph/main/install.ps1 | iex# 或通过 npm 全局安装npm install -g @colbymchenry/codegraph

安装后打开新终端使 PATH 生效

2 连接 Agent

codegraph install# 非交互模式codegraph install --target=cursor,claude --yes

支持 Claude Code / Cursor / Codex / opencode / Gemini / Kiro / Hermes / Antigravity

3 初始化项目

cd your-projectcodegraph init

创建 .codegraph/ 并构建图谱，之后自动同步

4 常用命令

# 一站式查询codegraph explore "how does auth work"# 查看调用者 / 影响范围codegraph callers UserService.logincodegraph impact UserService.login# 索引状态 / CI 集成codegraph statusgit diff --name-only | codegraph affected --stdin

5 更新 / 卸载

# 更新codegraph upgradecodegraph upgrade --check# 卸载（两步）codegraph uninstallcodegraph uninit --force

⚙ 可选配置

# 项目根目录创建 codegraph.json{"exclude":["static/"],"extensions":{".tpl":"php"}}# 修改后重建索引codegraph index

🔧 常见问题

"not initialized" → codegraph init

索引慢 → 确认 node_modules 已排除，加 --quiet

MCP 不连接 → codegraph status 或重跑 install

WSL2 失败 → 设 CODEGRAPH_NO_DAEMON=1

技术亮点

01

单工具哲学 codegraph_explore

一个 MCP 工具覆盖 90% 场景，减少 Agent 决策成本

02

三层保鲜：Watch → Debounce → Catch-up

文件监听 <100ms → 防抖 2s → 重连追赶同步

03

Tree-sitter + SQLite FTS5

语法级精准提取。Swift 编译器 25,874 文件 4 分钟索引

04

跨 Agent 共享索引

一份 .codegraph/ 服务所有 MCP Agent，切换零成本
