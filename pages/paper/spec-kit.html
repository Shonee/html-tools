完整使用指南

1 安装 Specify CLI

# 安装 uv 包管理器curl -LsSf https://astral.sh/uv/install.sh | sh# 安装 Spec Kit（替换版本号）uv tool install specify-cli --from git+https://github.com/github/spec-kit.git@vX.Y.Z# 验证安装specify --version

需 Python 3.11+、Git

2 初始化项目

specify init my-project --integration copilotcd my-project# 或在当前目录specify init . --integration claude

支持 copilot/claude/gemini/codex 等 30+ 集成

3 建立项目宪法

/speckit.constitution Create principles for code quality, testing, UX consistency

存入 .specify/memory/constitution.md

4 定义规范 + 规划

# 描述做什么和为什么（不涉及技术栈）/speckit.specify Build a photo album app...# 指定技术栈，生成实现计划/speckit.plan Use Vite + vanilla JS, SQLite

5 拆解任务 + 实现

/speckit.tasks/speckit.implement# 收敛检查（评估一致性）/speckit.converge

⚙ 升级与卸载

# 检查更新specify self check# 升级到最新版specify self upgrade# 指定版本specify self upgrade --tag v2.0.0# 卸载uv tool uninstall specify-cli

🔧 常见问题

uv 未安装 → curl -LsSf https://astral.sh/uv/install.sh | sh

Python 版本不够 → 需 3.11+，推荐用 pyenv 管理

找不到斜杠命令 → 确认已运行 specify init

Claude Code 集成 → --integration claude
