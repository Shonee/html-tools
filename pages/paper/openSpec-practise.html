完整使用指南

1 安装 OpenSpec CLI

# 全局安装 OpenSpecnpm install -g @fission-ai/openspec# 验证安装openspec --version

需 Node.js 18+

2 克隆实践仓库

git clone https://github.com/ForceInjection/OpenSpec-practise.gitcd OpenSpec-practise

含文档 + 示例代码 + 规范文件

3 运行 Node.js 示例

cd examples/ecommerce-mininpm install && npm test# 启动开发服务（端口 3000）npm start

零外部依赖，Node.js 内置测试运行器

4 运行 Python 示例

cd examples/ecommerce-mini-pythonpip install -r requirements.txt# 运行测试 + 启动服务pytestpython -m uvicorn src.api.server:app --reload

5 在自己项目中使用

# 在项目根目录初始化 OpenSpecopenspec init# 使用 AI 助手的 /opsx 命令/opsx:propose # 生成变更提案/opsx:apply # AI 按规范编码openspec archive # 归档规范

⚙ 学习路径

# 推荐阅读顺序1. docs/openspec-user-manual.md2. docs/openspec-practical-guide.md3. docs/openspec-ai-workflow-analysis.md4. examples/openspec/changes/v1-mvp/
