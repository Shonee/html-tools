解决什么问题

AI 编码助手理解项目时逐文件阅读，token 消耗巨大且容易遗漏关系。Graphify 把项目预构建为知识图谱（代码 AST 本地处理 + 文档 LLM 提取），AI 通过查询图谱获取精准上下文——token 消耗降低 71 倍，幻觉显著减少。

完整使用指南

1 安装

# macOS（推荐 uv 隔离环境）brew install uvuv tool install graphifyy# Windowswinget install astral-sh.uvuv tool install graphifyy

PyPI 包名 graphifyy（双 y），CLI 命令为 graphify

2 注册到 AI 助手

# Claude Code（默认）graphify install# 其他工具graphify install --platform codexgraphify cursor installgraphify install --platform gemini

3 构建知识图谱

# 在 AI 助手中输入/graphify .# 或指定目录/graphify ./src --mode deep

生成 graphify-out/（graph.html + GRAPH_REPORT.md + graph.json）

4 查询与使用

# 查询关系graphify query "auth 如何连接到数据库？"# 查找路径graphify path "UserService" "DatabasePool"# 解释概念graphify explain "RateLimiter"

5 团队协作与自动化

# 设置自动重建（Git hook）graphify hook install# 提交图谱供团队使用git add graphify-out/ && git commit# 增量更新（仅变更文件）/graphify . --update

6 更新与卸载

# 更新uv tool upgrade graphifyygraphify install# 卸载（所有平台）graphify uninstall --purge

🔧 常见问题

command not found → 运行 uv tool update-shell，重启终端

uvx 找不到包 → 用 uvx --from graphifyy graphify install

HTML 太大无法打开 → 加 --no-viz，用 graphify query 查询

可选扩展 → uv tool install "graphifyy[pdf,video,ollama]"
